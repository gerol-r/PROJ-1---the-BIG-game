Match It

A memory game.

Game Rules

  • Objective: Match all pairs of cards within 45 seconds.
  • Flipping Cards: Click two cards to flip them. Matches stay face up.
  • Winning: Match all pairs before the timer runs out.
  • Losing If time runs out before all pairs are matched, you lose.
  • Controls: Click the first card to begin and "Restart" button restart the game. Enjoy!
